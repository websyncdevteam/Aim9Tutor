# CBSE AI Classroom

An AI-powered tutor for CBSE classes 1-9 using Streamlit, Gemini 1.5 Pro, and Google TTS.

## Features
- Upload any CBSE textbook PDF (Class 1-9 only)
- Three modes: Teach, Quiz, Doubt Chat
- Natural voice output in English, Hindi, Hinglish, Bhojpuri
- Strict adherence to uploaded content
- Budget tracking and daily usage caps
